Kapal Bermuatan 35 Ton Minyak Tanah Terbakar di Perairan Mamoking Tulehu, Dua Orang Alami Luka Bakar

TERASMALUKU.COM,-AMBON-Sebuah kapal motor bermuatan BBM jenis minyak tanah terbakar di perairan Pelabuhan Speedboat di Dusun Mamoking, Desa Tulehu, Kecamatan Salahutu, Kabupaten Maluku Tengah, Jumat (21/10/2022) dini hari.


Kasi Humas Polresta Ambon, Ipda Moyo Utomo mengungkapkan akibat insiden nahas itu, dua orang korban alami luka bakar.

Kapal motor bernama Sayang Salsabila itu bermuatan 35 ton minyak tanah saat terbakar. Kapal ini berawakan Samsudin (50), Arobi (30), Malik (38) dan La Simina (64) selaku pemilik.

Saat kapal bermuatan 35 ton minyak tanah terbakar di perairan Mamoking Tulehu, kapal ini tengah berlabuh.

Menurut keterangan para saksi percikan api pemicu kebakaran yang berujung hangusnya kapal tersebut diduga berasal karburator mesin lampu di dek kapal.

Percikan api mucul saat mesin lampu dihidupkan Arobi beberapa saat pasca lampu kapal padam ketika ia dan rekan-rekannya tengah bekerja di ruang mesin kapal, Kamis (20/10/2022) tengah malam sekira pukul 23.00 WIT.

Karena api semakin besar dan menjalar ke semua badan kapal, Arobi an Malik terpaksa melompat ke laut untuk menyelamatkan diri.


Kasi Humas Polresta Ambon, Ipda Moyo Utomo.
“Pada saat itu juga api langsung menyala kemudian saksi (Arobi) bersama Malik mencoba untuk mematikan api dengan cara membasahi kain dan menutupi api tersebut, tetapi api tidak dapat dipadamkan sehingga api menjadi besar kemudian menjalar ke dek depan yang berisikan bahan bakar jenis minyak tanah sehingga api menjadi lebih besar dan langsung membakar habis kapal,”terangnya mengutip keterangan saksi.

Sedangkan saksi Samsudin dan La Simina langsung bergegas keluar dari kamar mesin dan menuju kapal yang berada disebalah kapal mereka itu memotong tali pengikat guna menghindari api menjalar ke kapal yang berlabuh sama-sama dengan Kapal Sayang Salsabilah


Akibat kejadian tersebut, La Simina dan Arobi alami luka bakar ringan pada kaki mereka. “Akibat kejadian kebakaran ini tidak menimbulkan adanya korban jiwa, namun terdapat luka ringan (luka bakar ringan di kaki) yang di alami oleh saudara La Simina dan Arobi,”tandasnya.

Disamping itu, kerugian materil juga dialami akibat muatan 35 ton mitan ikut terbakar bersama kapal.
