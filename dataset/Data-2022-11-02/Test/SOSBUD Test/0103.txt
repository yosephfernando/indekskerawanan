Tanah Longsor Ganggu Perjalanan Kereta Api di Kabupaten Malang
181020221401

Merdeka,com - Perjalanan kereta api terganggu akibat sejumlah titik pada lintasan Sumberpucung-Pohgajih di Kabupaten Malang, Jawa Timur, tertimbun tanah longsor, Selasa (18/10) pukul 09.08 WIB. PT Kereta Api Indonesia (Persero) Daerah Operasi 8 Surabaya masih berupaya memulihkan jalur itu.

Longsor menutupi jalur KA di KM 85+4/5 setelah sebelumnya daerah itu diguyur hujan lebat. Selain itu, terdapat tiga titik longsor yang berpotensi menutup jalur KA, yakni : KM 84+8/9, KM 82+6/7, dan KM 81+3 petak jalan Sumberpucung - Pohgajih.

"Kami atas nama Manajemen KAI mengucapkan permohonan maaf kepada para pelanggan karena terganggunya perjalanan dan pelayanan kereta api akibat jalur KA terkena longsor di wilayah tersebut," ujar Manager Humas KAI Daop 8 Surabaya Luqman Arif dalam keterangannya, Selasa (18/10).

Kereta Api Tertahan
KA Matarmaja yang berangkat dari Stasiun Malang menuju Pasarsenen keberangkatan pukul 09.25 WIB dilaporkan tertahan di Stasiun Sumberpucung. Kereta api itu rencananya dialihkan melewati Lawang-Bangil-Sidoarjo-Tulangan-Mojokerto lanjut Kertosono. Sementara KA Penataran relasi Surabaya-Blitar saat ini tertahan di Stasiun Kepanjen.

"Para pelanggan yang terdampak telah kami informasikan seluruhnya terkait kondisi terbaru perjalanan KA. Info selengkapnya terkait perjalanan KA, pelanggan dapat menghubungi Customer Service," tulis Luqman.

KAI Daop 8 Surabaya saat ini tengah melakukan upaya untuk memulihkan perjalanan kereta api di lokasi itu. Penanganan dilakukan baik dari sisi pengaturan operasional KA maupun prasarana jalur KA. KAI juga berkoordinasi dengan pihak-pihak terkait untuk mempercepat proses perbaikan.

"Kami berkomitmen untuk senantiasa mengutamakan keselamatan, keamanan, dan kenyamanan perjalanan KA. Semoga cuacanya mendukung dan longsor segera teratasi agar jalur kereta api yang terdampak kembali normal serta perjalanan kereta api dapat beroperasi dengan lancar," tutupnya.
